:py:mod:`osmg.gen.steel.brb`
============================

.. py:module:: osmg.gen.steel.brb

.. autoapi-nested-parse::

   Model Generator for OpenSees ~ BRB element generator

   The purpose of this component is to use the lower-level `gen` classes
   to simplify the definition of BRB elements simulated using a
   corotational truss opensees element and the Steel4 material wrapped
   under a Fatigue material, as done in [1] and [2].

   [1] Zsarnoczay, Adam. "Experimental and numerical investigation of
       buckling restrained braced frames for Eurocode conform design
       procedure development." (2013).
   [2] Simpson, Barbara Gwynne. Design development for steel strongback
       braced frames to mitigate concentrations of damage. University of
       California, Berkeley, 2018.

   .. py:method:: add_brb(xi_coord: float, yi_coord: float, lvl_key_i: int, offset_i: nparr, snap_i: str, xj_coord: float, yj_coord: float, lvl_key_j: int, offset_j: nparr, snap_j: str, area: float, f_y: float, e_0: float, casing_size: float, unit_weight: float) -> None

      Adds a BRB element to the model.

      :param xi_coord: x coordinate of the i-end
      :type xi_coord: float
      :param yi_coord: y coordinate of the i-end
      :type yi_coord: float
      :param lvl_key_i: Level ID of the i-end
      :type lvl_key_i: int
      :param offset_i: Rigid offset at the i-end
      :type offset_i: nparr
      :param snap_i: Snap tag for the i-end connectivity used to
                     automatically determine an offset based on existing
                     elements.
      :type snap_i: str
      :param xj_coord: x coordinate of the j-end
      :type xj_coord: float
      :param yj_coord: y coordinate of the j-end
      :type yj_coord: float
      :param lvl_key_j: Level ID of the j-end
      :type lvl_key_j: int
      :param offset_j: Rigid offset at the j-end
      :type offset_j: nparr
      :param snap_j: Snap tag for the j-end connectivity used to
                     automatically determine an offset based on existing
                     elements.
      :type snap_j: str
      :param area: Core area of the BRB element.
      :type area: float
      :param f_y: Steel stress.
      :type f_y: float
      :param e_0: Young's modulus, adjusted to account for
                  non-prismatic shape. See [2], Figure 4.15, and Equations
                  4.7, 4.8.
      :type e_0: float
      :param casing_size: Assuming a square cross-section,
                          controls the side length. Used for visualization purposes
                          and to determine the weight of the BRB element, which is
                          lumped to the connecting nodes. This parameter has no
                          effect on the force-deformation relationship of the
                          element.
      :type casing_size: float
      :param unit_weight: Weight per unit volume, used together
                          with the casing_size, assuming a square section and a
                          workpoint-to-workpoint length, in order to determine the
                          weight of the BRB, which is lumped to the two connecting
                          nodes of the Truss element as weight and mass when the
                          self-weight and self-mass preprocessing methods are
                          called.
      :type unit_weight: float
