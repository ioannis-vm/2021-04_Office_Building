.. py:class:: UIDGenerator

   Generates unique identifiers (uids) for various objects.

   .. py:method:: new(thing: str) -> int

      Generates a new uid for an object of the given type.

      :param object_type: The type of object for which to generate a uid.
      :type object_type: str

      :returns: A unique identifier for an object of the given type.
      :rtype: int

      .. rubric:: Example

      >>> from osmg.gen.uid_gen import UIDGenerator
      >>> generator = UIDGenerator()
      >>> generator.new('node')
      0
      >>> generator.new('node')
      1
      >>> generator.new('element')
      0
      >>> generator.new('element')
      1
