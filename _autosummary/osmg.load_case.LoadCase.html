osmg.load_case.LoadCase¶

class osmg.load_case.LoadCase(name: str, parent_model: Model, parent_nodes: dict[int, Node] = <factory>)¶

Bases: object

Load Case object. Load cases contain information related to the specified loads, mass, parent nodes and rigid diaphragm constraints, etc. Analysis objects can use multiple load cases. Load combination objects utilize load cases as well.
