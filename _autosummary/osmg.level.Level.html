osmg.level.Level¶

class osmg.level.Level(parent_model: Model, uid: int, elevation: float)¶

Bases: object

Level Object Levels are part of a model and they contain primary nodes and

component assemblies.

Examples

>>> from osmg.model import Model
>>> model = Model(name='example_model')
>>> level = Level(parent_model=model, uid=1, elevation=0.0)
>>> level.parent_model.name
'example_model'
>>> level.uid
1
>>> level.elevation
0.0
