osmg.solver.Results¶

class Results(periods: ndarray[Any, dtype[float64]] | None = None, n_steps_success: int = 0, metadata: dict[str, object] | None = None)[source]¶

Bases: object

Stores analysis results.

node_displacements¶

Displacements, stored in a dictionary. The keys correspond to the node UIDs. Each value is a dictionary, of which the keys are the analysis step. Each value of that is list containing the displacement of each DOF.

Type:

osmg.collections.Collection[int, dict[int, list[float]]]

node_reactions¶

osmg.collections.Collection[int, dict[int, list[float]]]

element_forces¶

Basic forces of beamcolumn elements. The nested structure is similar to that of node_displacements.

Type:

osmg.collections.Collection[int, dict[int, numpy.ndarray[Any, numpy.dtype[numpy.float64]]]]

release_foce_defo¶

Force-deformation pairs of zerolength elements. The nested structure is similar to that of node_displacements.

periods¶

Optional, stores the periods for modal analyses.

Type:

numpy.ndarray[Any, numpy.dtype[numpy.float64]] | None

int
