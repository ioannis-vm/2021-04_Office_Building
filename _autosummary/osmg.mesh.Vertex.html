osmg.mesh.Vertex¶

class Vertex(coords: tuple[float, float])[source]¶

Bases: object

2D Vertex. It knows all the edges connected to it. It knows all the halfedges leaving from it. Each instance has an automatically generated unique id.

coords¶

Coordinates of the vertex.

edges¶

List of edges connected to the vertex.

halfedges¶

List of halfedges leaving from the vertex.

uid¶

Unique identifier of the vertex.

Example

>>> from osmg.mesh import Vertex
>>> v = Vertex((0.0, 0.0))
