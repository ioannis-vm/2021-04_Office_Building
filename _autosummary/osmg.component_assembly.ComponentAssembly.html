osmg.component_assembly.ComponentAssembly¶

class ComponentAssembly(uid: int, parent_collection: Collection[int, ComponentAssembly], component_purpose: str)[source]¶

Bases: object

A component assembly represents some part of a structure and holds various lower-level elements such as nodes and beamcolumn elements.

uid¶

Unique identifier of the component assembly

Type:

int

parent_collection¶

The collection of elements to which the component assembly belongs.

Type:

osmg.collections.Collection[int, osmg.component_assembly.ComponentAssembly]

component_purpose¶

The functionality of the component assembly

Type:

str

external_nodes¶

the external nodes to which the component assembly is connected. these nodes should exist as part of a level.

Type:

osmg.collections.NodeCollection

internal_nodes¶

internal nodes that are required for the connectivity of the elements of the component assembly. these nodes only exist as part of the component assembly.

Type:

osmg.collections.NodeCollection

elements¶

Collection containing the elements that are part of the component assembly.

Type:

osmg.collections.CollectionWithConnectivity[int, osmg.ops.element.Element]

dict_of_elements()[source]¶

Returns a dictionary of all element objects in the model. The keys are the uids of the objects.

element_connectivity()[source]¶

Returns the connectivity of all elements. Elements are connected to external nodes. Each component assembly can be represented by a tuple of node uids of its connected nodes in ascending order. This method returns a dictionary having these tuples as keys, and the associated components as values.
