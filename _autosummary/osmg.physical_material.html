Physical material.
class PhysicalMaterial(uid: int, name: str, variety: str, density: float, e_mod: float, g_mod: float, f_y: float)[source]¶

Physical material. We use this for self-weight, plotting enhancements etc.

Example

>>> mat1 = PhysicalMaterial(
...     1, 'steel', 'structural',
...     7850, 210000, 81000, 400)
>>> print(mat1)
Physical material object
Name: steel
uid: 1
variety: structural
density: 7850
E: 210000
G: 81000
